01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수는 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수는 프로그래밍에서 데이터를 저장하고, 이 데이터를 필요에 따라 변경할 수 있는 공간을 의미한다. 변수에 저장된 데이터는 프로그램 실행 중에 변경될 수 있다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

변수에 데이터를 추가한다는 것은 기본적으로 변수의 현재 값에 새로운 값을 '더하거나' 기존 데이터 구조에 새로운 요소를 '추가하는' 작업을 말합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
400
600
javascriptjquery

04. 상수 : 데이터 저장 + 데이터 변경(X)

상수는 한 번 선언되고 초기화된 후에는 그 값을 변경할 수 없는 변수를 의미합니다. 프로그래밍에서 상수를 사용하는 주된 이유는 데이터의 안정성을 보장하고, 코드의 가독성을 높이며, 실수로 인한 값의 변경을 방지하기 위함입니다.
const를 사용한다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    // x = 300;
    // y = 400;
    // z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

05. 배열 : 데이터 저장(여러개) : 표현방법1 : 배열 선언

배열은 같은 타입의 데이터를 순차적으로 저장하는데 사용되는 데이터 구조입니다. []을 사용한다. 

{
    const arr = new (Array);

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(0);
    console.log(1);
    console.log(2);
}
결과 확인하기
100
200
javascript

06. 배열 : 데이터 저장(여러개) : 표현방법2 : 배열 선언과 동시에 초깃값 설정

배열을 생성함과 동시에 원하는 값으로 초기화할 수 있어, 코드의 가독성과 효율성을 높일 수 있습니다. 

{
    const arr = new Array(100, 200, "javascript");

    console.log(0);
    console.log(1);
    console.log(2);
}
결과 확인하기
100
200
javascript

07. 배열 : 데이터 저장(여러개) : 표현방법3 : 리터널 방식으로 배열 선언

배열 리터널은 빈 대괄호([])를 사용하여 빈 배열을 생성하는 것입니다. 이 방식을 통해 배열의 요소들을 바로 지정할 수 있습니다. 

{
    const arr = []

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

08. 배열 : 데이터 저장(여러개) : 표현방법5 : 리터널 방식으로 배열 선언과 동시에 초깃값 설정

데이터를 미리 배열에 저장하고자 할 때 사용됩니다. 배열 리터널은 대괄호([])를 사용하여 표현되며, 대괄호 안에 초기값을 쉼표(,)로 구분하여 나열함으로써 배열을 초기화할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

09. 배열 : 데이터 저장(여러개) : 표현방법4 : push() 메서드를 이용한 배열 방법

자바스크립트의 push 메서드는 배열의 끝에 하나 이상의 요소를 추가하고, 변경된 배열의 새로운 길이를 반환합니다. 동적으로 배열의 내용을 변경하고 싶을 때 매우 유용하게 사용됩니다. 

{
    const arr = [100, 200, "javascript"];

    arr.push(400);

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
    console.log(arr[3]);
}
결과 확인하기
100
200
javascript
400

10. 배열 : 데이터 저장(여러개) : 표현방법6 : 다차원 배열

다차원 배열은 배열 안에 또 다른 배열이 포함되어 있는 구조를 의미합니다. 가장 흔한 예로는 2차원 배열이 있으며, 이는 행렬(Matrix)과 유사한 형태로 데이터를 구성할 때 사용됩니다. 이러한 다차원 배열은 데이터를 테이블 형태로 구성하거나, 좌표계 데이터를 다루는 데 유용하게 사용될 수 있습니다. 

{
    const arr = [100, [200, 300], [400, 500, [600]], "javascript", "jquery"];

    console.log(arr[0]);
    console.log(arr[1][0]);
    console.log(arr[1][1]);
    console.log(arr[2][1]);
    console.log(arr[2][2][0]);
}
결과 확인하기
100
200
300
500
600

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열구조분해할당

자바스크립트에서 배열 구조 분해 할당(Destructuring assignment)은 배열의 값을 개별 변수에 쉽게 할당할 수 있게 해주는 표현식입니다. 배열 구조 분해 할당을 사용하면, 배열 내의 요소들을 변수에 직접 할당할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    const [a, b, c] = arr;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    const arr = [100, 200, [300, 400]];

    const [a, b, [x, y]] = arr;

    console.log(a);
    console.log(b);
    console.log(x);
    console.log(y);
}
결과 확인하기
100
200
javascript

100
200
300
400

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼침연산자

배열 펼침 연산자(Spread operator)는 ... 기호를 사용하여 배열이나 이터러블(iterable) 객체의 요소를 개별 요소로 펼쳐서 사용할 수 있게 해주는 자바스크립트의 표현식입니다. ",", "공백" 등을 간결하고 읽기 쉽게 만든다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1);                 
    console.log(...arr1);              
    console.log(...arr1, ...arr2);      
    console.log(...arr3);               
}
결과 확인하기
100, 200, javascript

100200javascript300400jquery
100200javascript500
