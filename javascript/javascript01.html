01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다. 변수는 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수는 프로그래밍에서 데이터를 저장하고, 이 데이터를 필요에 따라 변경할 수 있는 공간을 의미한다. 변수에 저장된 데이터는 프로그램 실행 중에 변경될 수 있다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

변수에 데이터를 추가한다는 것은 기본적으로 변수의 현재 값에 새로운 값을 '더하거나' 기존 데이터 구조에 새로운 요소를 '추가하는' 작업을 말합니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
400
600
javascriptjquery

04. 상수 : 데이터 저장 + 데이터 변경(X)

상수는 한 번 선언되고 초기화된 후에는 그 값을 변경할 수 없는 변수를 의미합니다. 프로그래밍에서 상수를 사용하는 주된 이유는 데이터의 안정성을 보장하고, 코드의 가독성을 높이며, 실수로 인한 값의 변경을 방지하기 위함입니다.
const를 사용한다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    // x = 300;
    // y = 400;
    // z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기

데이터를 미리 배열에 저장하고자 할 때 사용됩니다. 배열 리터널은 대괄호([])를 사용하여 표현되며, 대괄호 안에 초기값을 쉼표(,)로 구분하여 나열함으로써 배열을 초기화할 수 있습니다.

09. 배열 : 데이터 저장(여러개) : 표현방법4 : push() 메서드를 이용한 배열 방법

자바스크립트의 push 메서드는 배열의 끝에 하나 이상의 요소를 추가하고, 변경된 배열의 새로운 길이를 반환합니다. 동적으로 배열의 내용을 변경하고 싶을 때 매우 유용하게 사용됩니다. 

{
    const arr = [100, 200, "javascript"];

    arr.push(400);

    console.log(arr[0]);
    console.log(arr[1]);

    console.log(arr[3]);
}

10. 배열 : 데이터 저장(여러개) : 표현방법6 : 다차원 배열

다차원 배열은 배열 안에 또 다른 배열이 포함되어 있는 구조를 의미합니다. 가장 흔한 예로는 2차원 배열이 있으며, 이는 행렬(Matrix)과 유사한 형태로 데이터를 구성할 때 사용됩니다. 이러한 다차원 배열은 데이터를 테이블 형태로 구성하거나, 좌표계 데이터를 다루는 데 유용하게 사용될 수 있습니다. 

{
    const arr = [100, [200, 300], [400, 500, [600]], "javascript", "jquery"];

    console.log(arr[0]);
    console.log(arr[1][0]);
    console.log(arr[1][1]);
    console.log(arr[2][1]);
    console.log(arr[2][2][0]);
}

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열구조분해할당

자바스크립트에서 배열 구조 분해 할당(Destructuring assignment)은 배열의 값을 개별 변수에 쉽게 할당할 수 있게 해주는 표현식입니다. 배열 구조 분해 할당을 사용하면, 배열 내의 요소들을 변수에 직접 할당할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    const [a, b, c] = arr;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    const arr = [100, 200, [300, 400]];

    const [a, b, [x, y]] = arr;

    console.log(a);
    console.log(b);
    console.log(x);
    console.log(y);
}
결과 확인하기
100
200
javascript

200
300

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열펼침연산자

배열 펼침 연산자(Spread operator)는 ... 기호를 사용하여 배열이나 이터러블(iterable) 객체의 요소를 개별 요소로 펼쳐서 사용할 수 있게 해주는 자바스크립트의 표현식입니다. ",", "공백" 등을 간결하고 읽기 쉽게 만든다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1);                 
    console.log(...arr1);              
    console.log(...arr1, ...arr2);      
    console.log(...arr3);               
}
결과 확인하기
100, 200, javascript

18. 객체 : 데이터 저장(여러개) : 표현방법 6 : 객체 안에 배열이 있는 방식

19. 객체 : 데이터 저장(여러개) : 표현방법 7 : 객체 안에 객체가 있는 방식

20. 객체 : 데이터 저장(여러개) : 표현방법 8 : 객체 + 함수가 있는 방식

객체와 함수를 함께 사용하는 방식은 객체지향 프로그래밍에서 중요한 개념으로, 객체 안에 함수를 포함시켜 해당 객체의 동작을 정의하고 관리하는 방법을 말합니다 이를 통해 객체는 데이터와 동작을 함께 가질 수 있으며, 이를 통해 프로그램의 구조를 보다 모듈화하고 유연하게 만들 수 있습니다. 

{
    const obj = {
        a: 100,
        b: [200, 300],
        c: "javascript",
        d: function () {
            console.log("javascript가 실행되었습니다.");
        },
        e: function () {
            console.log(obj.c + "가 실행되었습니다.")
        },
        f: function () {             //this = 불러오기
            console.log(this.c + "가 실행되었습니다.")
        }
    }
    console.log(obj.a);
    console.log(obj.b[0]);
    console.log(obj.b[1]);
    console.log(obj.c);
    obj.d();
    obj.e();
    obj.f();
}
결과 확인하기

객체 구조 분해 할당(객체 디스트럭처링)은 JavaScript와 같은 언어에서 사용되는 기능으로, 객체의 속성을 해체하여 개별 변수에 할당하는 기술입니다. 이를 통해 객체의 속성을 추출하여 변수에 바로 할당할 수 있어서 코드를 보다 간결하고 가독성 있게 작성할 수 있습니다. 

{
