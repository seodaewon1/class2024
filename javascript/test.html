01. 1부터 100까지 출력하기(for문, while, do while)

for문을 이용해서 1-100까지 출력하는 예제입니다.

{
    //for
    {
        for (i = 1; i <= 100; i++) {
            console.log(i);
        }
    }
    //while
    {
        let i = 1
        while (i <= 100) {
            console.log(i);
            i++;
        }
    }
    //do while
    {
        let i = 1
        do {
            console.log(i)
            i++;
        }
        while (i <= 100)
    }
}
결과 확인하기
1,2,3,4 ...100

02. 0부터 100까지 짝수만 출력하기(for문, if문)

for문을 이용해서 0부터 100까지 짝수만 출력하는 예제입니다.

{
    for (let i = 0; i <= 100; i++) {
        if (i % 2 == 0) {
            console.log(i);
        }
    }
}
결과 확인하기
0, 2, 4, 6 ...100

03. 1부터 100까지 짝수는 파란색, 홀수는 빨간색 출력하기

for문을 이용해서 1부터 100까지 짝수는 파란색, 홀수는 빨간색을 출력하는 예제입니다.

{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            document.write("" + i + "");
        } else {
            document.write("" + i + "");
        }
    }
}
결과 확인하기
0, 2, 4, 6 ...100(blue)
1, 3, 5, 7 ...99(red)

04. 중첩 for문을 이용하여 구구단 출력하기

중첩 for문을 이용하여 구구단을 출력하는 예제입니다.

{
    for (let i = 2; i <= 9; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + "*" + j + "=" + i * j);
        }
    }
}
결과 확인하기
2*1=2, 2*2=4, ...2*9=18
3*1=3, 3*2=6, ...3*9=27
...9*9=81

05. 중첩 for문을 이용하여 8단 출력하기

중첩 for문을 이용하여 8단을 출력하는 예제입니다.

{
    for (let i = 8; i <= 8; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + "*" + j + "=" + i * j);
        }
    }
}
결과 확인하기
8*1=8, 8*2=16, ...8*9=72
