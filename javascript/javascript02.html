01. 변수 : 데이터 불러오기

자바스크립트에서 변수 데이터 불러오기는 변수에 저장된 데이터를 사용하는 과정을 말합니다. let, const, var를 사용하여 변수를 선언하고 데이터를 할당할 수 있습니다. 변수에 데이터를 저장하고, 필요할 때 그 데이터를 불러와서 사용할 수 있습니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100
200

05. 배열 : 데이터 불러오기 : for()문

for문은 프로그래밍에서 반복적인 작업을 수행할 때 사용되는 가장 기본적인 제어문 중 하나입니다. 특정 조건이 만족될 때까지 코드 블록을 반복 실행하게 해주며, 주로 배열이나 컬렉션의 모든 요소를 순회할 때 사용됩니다.

    }
}

10. 배열 : 데이터 불러오기 : filter()

자바스크립트의 filter 함수는 배열의 각 요소에 대해 주어진 함수를 실행하고, 그 결과가 true를 반환하는 모든 요소로 새 배열을 생성하는 메서드입니다. 이는 특정 조건을 만족하는 요소들만을 골라내고 싶을 때 유용하게 사용됩니다. 

{
    const num = [1, 2, 3, 4, 5, 6, 7, 8, 9];

    num.filter(function (element) {
        console.log(element);
    });

    num.filter(function (element, index) {
        console.log(index);
    });

    num.filter(function (element, index, array) {
        console.log(array);
    });

    const arr = num.filter(function (element) {
        return element > 5
    });
    console.log(arr);

    const arr1 = num.filter((element) => {
        return element > 5
    });
    console.log(arr1);

    const arr2 = num.filter(element => {
        return element > 5
    });
    console.log(arr2);

    const arr3 = num.filter(e => e > 5);
    console.log(arr3);
}
결과 확인하기

0, 1, 2, 3, 4, 5, 6, 7, 8

11. 배열 : 데이터 불러오기 : 배열 구조 분해 할당

배열 구조 분해 할당은 배열의 각 요소를 변수에 할당하는 간편한 방법을 제공합니다. Rest 연산자 사용: 배열의 일부 요소만 변수에 할당하고, 나머지 요소들을 다른 배열에 할당할 수 있습니다. 함수 매개변수에서의 사용: 함수를 호출할 때 배열 구조 분해 할당을 사용하여 매개변수를 전달할 수 있습니다. 

{
    const arr = [100, 200, "javascipt"];
    const [a, b, c] = arr;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 선언에서 분리한 할당
    const [a, b, c] = [100, 200, "javascipt"];

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 기본 값 설정
    const [a, b, c = "javascript"] = [100, 200];

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 배열 일부 요소 무시
    const [a, , b] = [100, 200, "javascript"];

    console.log(a);
    console.log(b);
}
{
    // 나머지 값 할당
    const [a, b, ...reset] = [100, 200, "javascript", "jquery"];

    console.log(a);
    console.log(b);
    console.log(...reset);
}
{
    // 변수 값 교환
    let a = 100, b = 200;
    [a, b] = [b, a];

    console.log(a);
    console.log(b);
}
{
    // 다차원 배열 펼침 연산자

    const [a, [b, c]] = [1, [2, "javascript"]];

    console.log(a);
    console.log(b);
    console.log(c);
}

12. 배열 : 데이터 불러오기 : 배열 펼침 연산자(Spread Operator)

배열 펼침 연산자는 ...로 표현됩니다. 이 연산자를 사용하면 배열, 문자열, 객체 등 반복 가능한 객체(Iterable Object)를 개별 요소로 분리할 수 있습니다.
배열 병합: 두 개 이상의 배열을 쉽게 하나로 병합할 수 있습니다.
함수 인자에서의 사용: 함수가 여러 개의 인자를 받을 때, 배열 펼침 연산자를 사용하여 배열의 요소를 개별 인자로 전달할 수 있습니다.
문자열에서의 사용: 문자열에 배열 펼침 연산자를 사용하면, 문자열의 각 문자를 배열의 요소로 분리할 수 있습니다.

{
        // 배열 복사
        const arr = [100, 200, "javascript"];
        const copy = [...arr];

        console.log(arr);
        console.log(...arr);
        console.log(copy);
    }
    {
        // 배열 합치기
        const arr1 = [100, 200];
        const arr2 = [300, 400];
        const merged1 = [...arr1, ...arr2];
        const merged2 = [...arr1, "javascript", ...arr2, "jquery"];

        console.log(arr1.concat(arr2));
        console.log(merged1);
        console.log(merged2);
    }
    {
        // 함수 인자로  사용하기
        function numbers(a, b, c) {
            return a + b + c;
        }
        const nums = [100, 200, 300];
        console.log(numbers(...nums));
    }
    {
        // 문자열 배열로 변환하기
        const text1 = "javascript";
        const text2 = [...text1];

        console.log(text1);
        console.log(text2);
    }
결과 확인하기

13. 객체 : 데이터 불러오기 : 기본

자바스크립트에서 객체는 키와 값의 쌍으로 이루어진 데이터 구조입니다. 객체 내의 데이터에 접근하는 방법은 크게 두 가지가 있습니다: 점 표기법(Dot notation)과 대괄호 표기법(Bracket notation)입니다.
•점 표기법(Dot notation): 가장 일반적인 방법으로, 객체의 속성에 접근할 때 사용됩니다..
•대괄호 표기법(Bracket notation): 속성 이름을 문자열로 지정해 접근합니다. 변수를 사용하여 동적으로 속성 이름을 지정할 때 유용합니다.

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(obj.a);
    console.log(obj.b);
    console.log(obj.c);

    console.log(obj["a"]);
    console.log(obj["b"]);
    console.log(obj["c"]);
}
결과 확인하기
100
200
javascript

17. 객체 : 데이터 불러오기 : Object.assign

여러 객체를 하나의 객체로 병합할 때 Object.assign을 사용할 수 있습니다. 소스 객체의 속성이 대상 객체의 속성과 중복될 경우, 소스 객체의 값으로 대체됩니다. Object.assign을 사용하여 객체를 복사할 수 있습니다. 

{
    const obj1 = {
        a: 100,
        b: 200,
    }
    const obj2 = {

19. 객체 : 데이터 불러오기 : for in

for...in 반복문은 객체의 모든 열거 가능한 속성에 대해 반복하며, 각 속성의 키(또는 이름)을 통해 작업을 수행할 수 있게 해줍니다. 이 반복문은 객체 내부의 속성을 탐색할 때 유용하게 사용됩니다. 대괄호 표기법을 사용하여 객체의 속성 값을 접근합니다. 여기서 el은 객체 obj의 각 속성 이름을 나타냅니다. 

{
    const obj = {

        c: "javascript"

    }
}

21. 객체 : 데이터 불러오기 : 객체 펼침연산자

객체 펼침연산자는 ES6에서 도입된 JavaScript의 유용한 기능 중 하나로, 마침표 세 개(...)를 사용하여 객체나 배열의 요소를 개별적으로 분리하거나, 여러 객체를 하나로 합치는 등의 작업을 간단하게 할 수 있게 해줍니다. 

{
    const obj1 = { a: 100, b: 200 };
    const obj2 = { c: 300, d: 400 };
    const merge1 = { ...obj1, ...obj2 };
    const merge2 = { ...obj1, x: "javascript", ...obj2, y: "jquery" };

    console.log(obj1);
    console.log(merge1);
    console.log(merge2);
}

{a: 100, b: 200, x: javascript, c: 300, d: 400, y: jquery}

22. 서버 : 데이터 불러오기 : XMLHttpRequest

XMLHttpRequest(XHR)는 웹 페이지가 사용자의 개입 없이 서버와 데이터를 교환할 수 있게 해주는 JavaScript 객체입니다. 이를 통해 웹 페이지는 페이지의 새로고침 없이도 서버로부터 데이터를 받아와서 동적으로 내용을 변경할 수 있습니다. 

{
    var xhr = new XMLHttpRequest();
    xhr.open("GET", "https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json", true);

    xhr.onreadystatechange = function () {
        if (xhr.readyState === 4 && xhr.status === 200) {
            var responseData = xhr.responseText;
            var jsonData = JSON.parse(responseData);

            console.log(jsonData);
        } else if (xhr.readyState === 4) {
            console.error("데이터 불러오기 오류: " + xhr.status);
        };
    };
}

Fetch API는 자바스크립트를 사용하여 네트워크 요청을 보내고 받는 간단하면서도 강력한 방법을 제공합니다. 기본 사용법: Fetch 함수는 최소한 하나의 인자(자원의 경로)를 받으며, 선택적으로 설정 객체를 두 번째 인자로 받을 수 있습니다.
