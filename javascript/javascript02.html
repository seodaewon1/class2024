01. 변수 : 데이터 불러오기

자바스크립트에서 변수 데이터 불러오기는 변수에 저장된 데이터를 사용하는 과정을 말합니다. let, const, var를 사용하여 변수를 선언하고 데이터를 할당할 수 있습니다. 변수에 데이터를 저장하고, 필요할 때 그 데이터를 불러와서 사용할 수 있습니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100
200
javascript

02. 상수 : 데이터 불러오기

자바스크립트에서 상수 데이터 불러오기는 const 키워드를 사용하여 선언된 상수에 저장된 데이터를 사용하는 과정을 말합니다. 상수는 한 번 할당되면 그 값이 변경되지 않는 변수를 의미합니다. 

{
    const x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}
결과 확인하기
100
200
javascript

03. 배열 : 데이터 불러오기

자바스크립트에서 배열 데이터 불러오기는 배열 내에 저장된 여러 데이터 중 원하는 데이터를 접근하고 사용하는 과정을 말합니다. 배열은 여러 데이터를 순서대로 저장하는 자료구조로, 각 데이터는 인덱스를 통해 접근할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0], arr[1], arr[2]);
}
결과 확인하기
100
200
javascript

04. 배열 : 데이터 불러오기 : 갯수

자바스크립트에서 배열의 데이터 개수를 불러오는 것은 배열 내에 저장된 요소의 총 수를 확인하는 과정입니다. 이는 배열을 다룰 때 매우 기본적이면서도 중요한 작업 중 하나로, 배열의 길이를 알아내는 데 사용됩니다. 배열의 length 속성을 사용하면 배열에 저장된 요소의 총 개수를 쉽게 알 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr.length);
}
결과 확인하기
3

05. 배열 : 데이터 불러오기 : for()문

for문은 프로그래밍에서 반복적인 작업을 수행할 때 사용되는 가장 기본적인 제어문 중 하나입니다. 특정 조건이 만족될 때까지 코드 블록을 반복 실행하게 해주며, 주로 배열이나 컬렉션의 모든 요소를 순회할 때 사용됩니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < arr.length; i++) {      //초깃값; 조건식; 증감;
        console.log(arr[i]);
    }
}

06. 배열 : 데이터 불러오기 : forEach

JavaScript의 forEach 메서드는 배열을 순회하는 데 사용되는 메서드 중 하나입니다. 이 메서드는 배열의 각 요소에 대해 주어진 함수를 한 번씩 실행합니다. forEach는 배열의 요소들을 반복하여 작업을 수행할 때 유용하며, 함수형 프로그래밍 패러다임을 따르는 코드를 작성할 때 자주 사용됩니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    // 01
    arr.forEach(function (element) {
        console.log(element);
    })
    // 02
    arr.forEach(function (element, index) {
        console.log(index);
    })
    // 03
    arr.forEach(function (element, index, array) {
        console.log(array);
    })

    // 화살표함수
    // 01
    arr.forEach((element) => {
        console.log(element);
    })
    // 02
    arr.forEach(element => {
        console.log(element);
    })
    // 03
    arr.forEach(e => console.log(e));
}

09. 배열 : 데이터 불러오기 : map()

JavaScript의 map() 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하고, 그 결과로 새로운 배열을 생성합니다. 이 메서드는 기존 배열을 변형하지 않으며, 대신 각 요소에 대한 함수의 호출 결과를 요소로 하는 새 배열을 반환합니다. map()은 데이터의 변환, 가공 등에 유용하게 사용됩니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    arr.map(function (element) {
        console.log(element);
    })

    arr.map(function (element, index) {
        console.log(index);
    })

    arr.map(function (element, index, array) {
        console.log(array);
    })
}
